📜 Términos y Condiciones de Uso

1. Validación urbana

Esta plataforma permite conectarse a la red Wioo validando un comprobante de pago o un código institucional entregado por colaboradores autorizados.

2. Uso ético y responsable

Está prohibido compartir códigos, manipular el flujo de validación o utilizar credenciales que no le correspondan. El usuario se compromete a respetar los espacios digitales públicos y privados donde opera el sistema.

3. Protección de datos

Los datos ingresados solo se usan para fines de validación y nunca se comparten ni almacenan fuera del sistema urbano. La plataforma aplica políticas estrictas de privacidad.

4. Condiciones de navegación

La red Wioo se ofrece mediante infraestructura satelital, garantizada técnicamente bajo condiciones climáticas normales. El tiempo de conexión depende del comprobante validado.

5. Advertencia técnica

La conectividad puede verse afectada por fenómenos naturales como lluvias intensas, tormentas eléctricas o alta interferencia atmosférica. Estas situaciones podrían interrumpir temporalmente el servicio sin que ello represente una falla del sistema urbano.

6. Normas urbanas complementarias

El acceso no otorga derecho a alterar configuraciones, afectar redes vecinas ni interferir con otros usuarios. Cualquier uso indebido podrá ser reportado y gestionado institucionalmente.
